# Stock Market Prediction using Technical Indicators

## Overview

This project predicts stock market trends using historical market data and technical indicators. Machine learning techniques were applied to analyze market behavior and forecast future price movements.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Technical Indicators

- Moving Averages (MA)
- Relative Strength Index (RSI)
- MACD
- Bollinger Bands

## Features

- Data preprocessing
- Feature engineering
- Technical indicator calculation
- Market trend prediction
- Model evaluation

## Applications

- Financial data analysis
- Stock market forecasting
- Algorithmic trading research

## Dataset Availability

The original dataset used during experimentation is not included in this repository. Users may use publicly available stock market datasets (e.g., Yahoo Finance historical data) to reproduce the workflow.)
